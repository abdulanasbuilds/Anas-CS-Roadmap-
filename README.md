🚀 Universal CS Mastery Roadmap

A complete, self-paced, 100% free computer science curriculum — from zero to professional.

Built with resources from Harvard, MIT, Stanford, Google, and more, this roadmap is designed for anyone who wants to master computer science, software engineering, AI, IT, or cybersecurity without spending a single cedi or dollar. No gatekeeping. No paywalls. Just the best free education the world has to offer — structured into a clear, actionable path.

---

✨ Why This Exists

A world-class CS education shouldn't cost thousands. Every course, certificate, and lab linked here is genuinely free — taught by the same professors who teach at the world's top universities. This roadmap stitches them together into one coherent journey so you never wonder "what should I learn next?"

---

🧭 The 7 Phases

Phase Focus Duration
01 Computational Thinking & CS Foundations — Harvard CS50x, MIT 6.001 1–3 months
02 Programming Mastery & Software Engineering — MIT Python, Odin Project, Algorithms 2–4 months
03 IT, Systems & Networking — Google IT Cert, Linux Foundation, Cisco 1–2 months
04 Cybersecurity — ISC2, Google Cyber Cert, TryHackMe, Harvard CS50 Cyber 3–5 months
05 Mathematics for CS — MIT Calculus & Linear Algebra, Harvard Statistics 2–3 months
06 AI & Machine Learning — Stanford CS229, DeepLearning.AI, Fast.ai, Microsoft AI 4–6 months
07 Build, Deploy & Go Professional — GitHub portfolio, Kaggle, advanced electives Ongoing

🔥 Features of the Roadmap Page

· Progress tracking — Click checkboxes to mark steps complete. Progress saves to your browser (localStorage).
· 40+ curated steps — Each with descriptions, direct resource links, skills tags, and pro tips.
· Resource badges — Instantly see what's free, what awards a certificate, and what's a lab or YouTube companion.
· Dark cyberpunk aesthetic — Clean grid background, neon green accents, monospace typography.
· Phased accordion layout — Collapse phases you're not working on. Open the first one by default.
· Mobile responsive — Works on phones, tablets, and desktops.
· Single HTML file — No dependencies, no build tools. Open it anywhere.

---

🚀 Getting Started

1. Clone or download this repository.
   ```bash
   git clone https://github.com/abdulanasbuilds/cs-roadmap.git
   ```
2. Open cs-roadmap.html in your browser. That's it.
3. Start Phase 01 — Harvard CS50x is your first stop. The page links you directly there.
4. Track your progress by clicking the checkboxes. Your progress stays saved even if you close the tab.
